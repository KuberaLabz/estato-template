# Estato — Luxury Real Estate Template

A cinematic, editorial HTML/CSS/JS template for luxury real estate firms. Designed around the "quiet luxury" aesthetic — warm parchment, deep forest green, brushed gold. Built for the kind of firm that treats every home as a work of art.

![Design](https://img.shields.io/badge/Design-Quiet_Luxury-0E0C0A?style=flat-square)
![AI](https://img.shields.io/badge/AI-Vapi_Concierge-C9A96E?style=flat-square)

---

## ✨ Signature Features

- 🖱️ **Custom animated cursor** — dot + ring with lerp smoothing, dark/light mode auto-switch
- 🏛️ **Cinematic hero** — architectural line art, full-bleed dark background, property preview card
- 🏠 **Interactive listing cards** — hover reveals desc + CTA, click opens full property modal
- 📊 **Animated stats** — counters animate from zero on scroll
- 🤖 **AI Concierge** — Vapi chat widget + voice button (dark theme, gold accent)
- 📱 **Fully mobile responsive** — hamburger drawer, all sections reflow perfectly
- 🔒 **Privacy Policy + Terms** — complete legal modals, all footer links functional
- ⚡ **One config file** — all content, listings, team, and settings in `config.js`

## 🎨 Design System

| Token | Value | Use |
|-------|-------|-----|
| `--parchment` | `#F5F0E8` | Main background |
| `--forest` | `#2D3B2F` | Dark accent sections |
| `--near-black` | `#0E0C0A` | Hero, approach, contact |
| `--gold` | `#C9A96E` | Primary brand accent |
| `--serif` | Cormorant Garamond | Headlines (ultra-luxury) |
| `--sans` | Inter | Body, labels, UI |
